# Image prompts

## Consistency contract

Warm storybook fantasy, gouache/watercolour digital painting, natural light, bold silhouettes, odd memorable details. No baked-in text. No franchise lookalikes.

## Negative prompt (baseline)

`text, watermark, logo, photorealistic, 3d render, plastic toy, neopets, pokemon, harry potter, borderlands cel shading, grey placeholder, UI mockup, gibberish signs`

## Examples

**Spriggle:** cream fluffy companion, unusual soft ears, tiny green backpack, living sprout on head, storybook gouache, soft lighting, plain background

**Backwards Town:** warm crooked fantasy town square, fountain, timber houses, gentle surreal time motif, painted backdrop, no readable sign text

**Cracked moon overlay:** pale moon with a single visible crack, transparent sky, painterly, no text
